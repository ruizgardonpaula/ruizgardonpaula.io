# 💻 Paula Ruiz | Portfolio de Ingeniería Informática

Bienvenido/a a mi portfolio profesional. Este proyecto es una Single Page Application (SPA) diseñada para mostrar mi trayectoria, formación y stack tecnológico en las áreas de **Ciberseguridad**, **IA** y **Desarrollo de Software**.

🌐 **Puedes ver el portfolio en vivo aquí:** [ruizgardonpaula.github.io](https://ruizgardonpaula.github.io/ruizgardonpaula/)

---

## 🚀 Sobre el Proyecto

El objetivo de este sitio es presentar una interfaz limpia, moderna (estilo Apple/Minimalista) y totalmente responsiva que facilite a los reclutadores y colaboradores técnicos conocer mi perfil de un vistazo.

### Características principales:
- **Diseño Apple Style:** Uso de desenfoques (backdrop-filters), tipografía Plus Jakarta Sans y bordes redondeados.
- **Bento Grid:** Organización moderna de aptitudes técnicas para una lectura rápida.
- **Storyline Interactiva:** Sección de experiencia profesional con línea de tiempo visual.
- **Totalmente Responsivo:** Optimizado para dispositivos móviles, tablets y escritorio.

---

## 🛠️ Stack Tecnológico

Este portfolio ha sido desarrollado utilizando tecnologías web estándar para garantizar la máxima compatibilidad y velocidad de carga:

* **HTML5** - Estructura semántica.
* **CSS3** - Diseño basado en variables (Custom Properties), Flexbox y CSS Grid.
* **Font Awesome** - Iconografía profesional.
* **Google Fonts** - Tipografía personalizada (Plus Jakarta Sans & JetBrains Mono).
